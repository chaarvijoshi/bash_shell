# Bash Shell Scripting

A collection of beginner-friendly Bash shell scripting examples covering variables, user input, loops, functions, conditional statements, and command-line arguments.

## 📂 Repository Structure

| Script | Description |
|---------|-------------|
| `arg.sh` | Demonstrates how to use command-line arguments. |
| `bash.sh` | Basic Bash script example. |
| `check.sh` | Performs conditional checks. |
| `function.sh` | Demonstrates creating and calling functions. |
| `if-else.sh` | Shows how to use if-else statements. |
| `input.sh` | Accepts user input. |
| `loop.sh` | Demonstrates loop structures (`for`, `while`). |
| `silent_input.sh` | Reads hidden input such as passwords. |
| `useradd.sh` | Example of adding a user using Bash commands. |
| `var.sh` | Demonstrates variable declaration and usage. |
| `var2.sh` | Additional examples of variables. |

## 🚀 Prerequisites

- Linux or macOS
- Bash Shell

Check your Bash version:

```bash
bash --version
```

## ▶️ Running the Scripts

Give execute permission:

```bash
chmod +x filename.sh
```

Run the script:

```bash
./filename.sh
```

or

```bash
bash filename.sh
```
